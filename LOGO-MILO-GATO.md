## Código Actual Implementado

### **HTML (Iconos):**
```html

```

### **CSS (Animaciones):**
```css
.fa-cat {
    color: #ff6b35;
    animation: catWag 2s ease-in-out infinite;
}

@keyframes catWag {
    0%, 100% { transform: rotate(0deg); }
    25% { transform: rotate(-3deg); }
    75% { transform: rotate(3deg); }
}
```
